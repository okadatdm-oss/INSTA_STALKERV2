# INSTA_STALKER V2

Ferramenta simples para buscar menções públicas de perfis do Instagram.

Compatível com:

- iSH
- Alpine Linux
- Linux

---

# Instalação

apk update
apk add git

git clone https://github.com/okadatdm-oss/INSTA_STALKERV2.git

cd INSTA_STALKERV2

sh install.sh

sh insta_stalker.sh
